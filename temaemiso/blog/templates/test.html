Python環境のセットアップ概要

Pythonの環境のセットアップには以下のソフトウェアのインストールが必要です。
失敗しないように一つずつ確実にインストールしていきます。

  • Python
  • pip
  • pipx
  • poetry

Python環境のセットアップ手順

1. Pythonのインストール

how-to-install-from-the-full-installer に従いインストールを行う

環境変数の設定の仕方がわからないときは有識者に確認してください。

2. pipのインストール

以下のコマンドを実行（詳細はこちら）

py -m ensurepip --upgrade
Tips

基本的にpipはPythonをインストールした際に一緒にインストールされます
Requirement already satisfied: pip in ...
のような文言が表示されている場合はインストール済みですので、
次の手順に移ってください。

3. pipxのインストール

以下のコマンドを実行（詳細はこちら）

span>py -m pip install --user pipx

依存関係のあるソフトウェアのダウンロードが走る

最後に以下のような文言が出ていればOK

WARNING: The script pipx.exe is installed in <USER folder>\AppData\Roaming\Python\Python3x\Scripts which is not on PATH

...\AppData\Roaming\Python\Python3x\Scripts\Scriptsの部分をコピーし
以下のコマンドを実行

cd ...\AppData\Roaming\Python\Python3x\Scripts\Scripts

lsコマンドなどでpipx.exeがあることを確認し、以下のコマンドを実行

.\pipx.exe ensurepath

4. poetryのインストール

以下のコマンドを実行（詳細はこちら）

pipx install poetry
    
span>py -m pip install --user -U pipx
最後に以下のような文言が出ていればOK

done! ✨ 🌟 ✨

次のステップ

上記までで、python環境の構築は完了です！
プロジェクト名などが決まっている方は
次のpoetry環境の構築までやってしまいましょう！

1. Poetryプロジェクトの作成

まだディレクトリがない場合は以下を実行
※my-projectは任意のプロジェクト名に変更してください。

poetry new my-project

既存のディレクトリをpoetry環境化する場合は以下のコマンドを実行

poetry init
必ずPoetry環境化したいディレクトリに移動してから実行してください

コマンド実行時にいろいろ尋ねるプロンプトが出てきますが、
すべてEnterを押してください。

実行完了後 poetry.toml というファイルが生成されます。

2. 依存関係の追加 and インストール

プロジェクトに必要な依存関係を追加
※以下はFlaskをインストールする例です。

poetry add flask
Tips

「依存関係ってなに？」と思った方は、
現時点では「Poetryで作成した仮想環境にライブラリを追加する」
というイメージを持っておくとよいです。

依存関係をインストールpoetry環境へインストール

poetry install
